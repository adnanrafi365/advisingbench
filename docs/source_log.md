# UIC Official Source Log

This file tracks official public UIC sources used in AdvisingBench.

## Source Collection Rules

- Use only public official UIC pages.
- Do not use login-protected pages.
- Do not collect private student records.
- Do not collect advisor emails or personal student data.
- Store source title, URL, category, and date accessed.
- Use citations in the dataset and report.
